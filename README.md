## Development

1. Clone this repository.
2. Run `npm i`.
3. Run `npm run watch` to start the dev server..
4. Access the dev site at <http://localhost:3001>.

The site will hot-reload as you make changes.

## Building

1. Run `npm build` to build the site.
2. The built assets will be located in the `dist` folder.
